# skrimpers-platform
AI-powered Swiss price intelligence platform. 9 grocery retailers, 130K+ products, 6K+ fuel stations (CH + 100km cross-border into DE/FR/IT/AT), 18K+ EV charging stations. 4 languages, daily automated pipelines. Full-stack: Scrapy/ECS Fargate, medallion ETL, DynamoDB, React 19 + Cognito auth. Solo-built with AWS and AI. https://skrimpers.com
